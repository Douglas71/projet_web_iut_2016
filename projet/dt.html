Introduction

Lors de ce projet, nous avons procédé en plusieurs étapes :

  1. Brainstorming
  2. Conception de la maquette sur papier
  3. Réalisation de la page d'accueil (index.html)
  4. Réalisation des autres pages avec adaptation et rajout en CSS
  5. Tests finaux

Création de la maquette & Brainstorming

Nous avons passés au moins une heure pour penser le site avant de commencer à coder. La forme, la charte graphique, les images, le thème, etc...
Nous avons ensuite mis tout cela sur papier, en faisant des schémas en s'assurant que tout répondait bien aux consignes.

Réalisation de la page d'accueil

Cette page est le coeur du projet : toute la structure est répétée sur chaque autre page. C'est la page qui nous a demandé le plus de travail. Nous avons pu constater que certaines idées sur papier n'étaient pas forcément bonnes ou implémentables sur le site.
Nous avons commencé par réaliser la structure HTML, puis ensuite nous avons créé le CSS et implémenté les images. A chaque étapes, nous avons validé le code CSS et HTML avec le valideur W3C. Une fois la page d'accueil terminée, nous avons créé les autres pages. Une fois tout cela fait, nous avons adapté le site en responsive, via les medias queries.

Réalisation des autres pages

Les autres pages ont été réalisées sur le modèle de la page d'accueil, avec quelques modifications pour s'adapter à chaque spécificité de chaque page.

Tests finaux & mise en production

Encore quelques bugs corrigés (responsive, différents navigateurs, etc...), puis rédaction et correction des fautes d'orthographes.
Aujourd'hui, lundi 21/03/2016, le site est officielement mis en production.
